# 鼠须管98五笔

- 法一：假如您有 git 以及 gitee

打开终端输入

1. 使用 github
``````
git clone https://github.com/98wb/mac.git
cd squirrel
sh install.sh
``````



2. 使用 gitee

``````
git clone https://gitee.com/wubi98/squirrel.git
cd squirrel
sh install.sh
``````

- 法二：假如您没有 git

下载并解包，终端到 install.sh 所在目录

``````
sh install.sh
``````

- 法三：假如您啥也没有

去 [98五笔资源库](http://98wb.ys168.com/) 下载「鼠须管程序.zip」

解包，照图操作。
